# これは何？
プログラムワークショップ4の管理用

# 使い方

- 本リポジトリ(tpu-game-2023/PGWS4)をforkしてください。
- fork先のリポジトリを更新してください
- 結果を画面キャプチャして、下記のリンクで見れるようにしてください
- 完成したら本リポジトリのmainブランチにpull requestを投げてください
